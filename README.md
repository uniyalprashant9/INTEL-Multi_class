# Classification on INTEL-Multi_class dataset 

## The dataset has 6 classes as follows
- Building :office:
- Street :city_sunrise:
- Forest :evergreen_tree:
- Glacier :snowflake:
- Mountain :mount_fuji:
- Sea :ship:

## Orange Modules Used
- Image Loader
- Image Viewer
- Image Embedding
- Data Table
- Data Sampler
- Classification Algorithms
- Prediction
- Save Model
- Load Model
- Confusion Matrix
